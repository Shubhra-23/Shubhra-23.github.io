Artifact 01
AI Interviewer
Introduction

AI Interviewer is a conversational chatbot built to help job seekers practice and prepare for interviews. Users provide a job description and their resume, and the bot generates tailored interview questions, evaluates responses, and offers constructive feedback—simulating a real interview environment powered by large language model technology.

Description
Objective
To design and deploy an AI-powered interview preparation tool that personalizes the mock-interview experience by aligning questions to a specific job role and the candidate's background—reducing interview anxiety and improving readiness.
Process
The chatbot was designed using an iterative, prompt-engineering approach. The workflow began with defining user personas and requirements, followed by crafting system prompts that guide the LLM to act as a professional interviewer. Multiple rounds of testing and refinement were performed to improve question quality, feedback tone, and conversation flow. The final build was deployed as a publicly accessible artifact.
Tools & Technology Used
The following tools were used to build, test, and host this artifact:
Claude (Anthropic) Prompt Engineering GitHub HTML / CSS / JS LLM API
Value Proposition of the Artifact
Unique Value
Unlike generic interview prep resources, this chatbot dynamically tailors every question to the specific job description and user's resume. It creates a personalized, interactive feedback loop that static guides or flashcards cannot replicate, making interview preparation faster, more relevant, and more effective.
Relevance
As AI tools become increasingly central to workforce development and HR technology, this artifact demonstrates practical application of LLMs in a high-impact, real-world scenario. It is directly relevant to the AIML-500 course objectives around applied AI, human-centered design, and responsible AI tool development.
